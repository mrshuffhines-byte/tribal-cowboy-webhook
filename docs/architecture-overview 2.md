# Tribal Cowboy Social Media Operating System — Architecture Overview

This document explains how the full system works: who the agents are, how they interact, how content moves from idea to published post, and how every part of the operation connects back to the website and to creator income.

---

## System Architecture

### The Big Picture

The Tribal Cowboy social media operating system is a 13-agent network designed to run a professional content operation in 30 minutes or less per day. The system is built around one principle: every piece of content should do more than one job simultaneously — build audience trust, signal creator credibility, support bookings, and be evaluable for monetization, all at once.

The system is not a content factory. It is a decision-making framework that ensures the right content gets made for the right reasons, reviewed through the right lens, distributed in the right format, and followed up on in the right way.

### Who Is in Charge

The Orchestrator Agent is the hub of the system. It does not produce content — it directs traffic. Every input (a request from Stacie, a content idea, a campaign brief, an inbound message) goes through the Orchestrator first. The Orchestrator reads the input, determines which agent or agents need to be activated, passes the work in the right order, and holds the outputs until they are ready for Stacie's review.

### How Work Flows

**Idea to post, simplified:**

1. Input arrives (Stacie has an idea, an event happened, a seasonal moment needs content)
2. Orchestrator routes it to the Content Strategy Agent for pillar and placement decisions
3. Content Strategy passes to the Draft Writer Agent (Instagram, Facebook, or TikTok, depending on format)
4. Draft moves to Brand Voice Agent for review against TC's voice and non-negotiable rules
5. Brand Voice passes to the Content Calendar Agent for scheduling
6. Calendar Agent queues the post with a recommended time and platform
7. Stacie reviews and approves (or sends back with notes)
8. Post goes live
9. Engagement Routing Agent monitors inbound responses and classifies them
10. Follow-Up Agent generates reply drafts for DMs and comments
11. Performance Signal Agent logs observable patterns and flags anything worth noting

Parallel to this main flow, the Creator Income Agent and the Corporate/Partnership Signal Agent evaluate every piece of content before it posts — not to change it, but to flag opportunities it creates.

## Agent Directory

### 1. Orchestrator Agent

The central coordinator. Every request, task, idea, or inbound message enters the system through the Orchestrator. It reads context (current season, recent content, active campaigns, current priorities), determines which agents to activate, sequences the work correctly, and delivers outputs to Stacie in consolidated form rather than piecemeal. The Orchestrator does not write content, make voice judgments, or handle scheduling — it runs the room so everything else works without Stacie having to manage multiple moving pieces. Use this agent whenever you need the system to handle something from start to finish without specifying every step yourself.

### 2. Content Strategy Agent

Responsible for deciding what to make and why. When Stacie has an idea, a photo, an event recap, or a campaign need, the Content Strategy Agent determines which content pillar it belongs to, what platform it should live on, what format it should take, and what secondary objectives it can serve simultaneously. It keeps track of pillar rotation so the same type of content doesn't post back to back. It also flags when a content gap exists — when a pillar hasn't been touched in too long or when the booking-to-non-booking ratio is getting out of balance. Use this agent when planning a week's content, building a campaign, or asking "what should I post about?"

### 3. Draft Writer Agent (Instagram)

Writes Instagram captions to the TC brand standard. It knows the platform's tone — warm, visual, a little story — and applies the voice rules automatically: no exclamation mark overuse, no clichés, no AI-speak, no corporate language, specificity over generality. It writes the caption and selects hashtags from the hashtag bank based on the content. It also recommends optimal posting time based on the day and content type. Use this agent whenever you need an Instagram caption written or revised.

### 4. Draft Writer Agent (Facebook)

Writes Facebook posts to a slightly different tone — community-forward, slightly longer, good for event proof and local audience connection. Facebook readers here are families and community members who may already know the TC name. This agent leans into event recaps, behind-the-scenes access, and local-facing language. It uses 2–3 hashtags maximum and always connects content to something the audience in North Idaho and Eastern Washington can act on or engage with. Use this agent for all Facebook post creation.

### 5. Draft Writer Agent (TikTok)

Writes TikTok scripts and hooks for both the TC Business account and Stacie's personal creator account. It knows the difference between the two: Business content is hook-led and retention-focused, designed to grow reach and signal brand presence. Personal content is creator-personality content — Stacie talking to camera, behind-the-scenes narration, honest takes on the business. This agent writes the first three lines of every TikTok first, because the hook determines whether the rest of the video matters. Use this agent for TikTok scripts, Reel concepts, and short-form video outlines.

### 6. Brand Voice Agent

The gatekeeper. Every piece of drafted content passes through this agent before it goes anywhere. It checks for: corporate language, banned phrases, overuse of punctuation, AI-speak, clichés, and any language that makes Stacie sound like a marketing department rather than a person. It also checks that Indigenous identity is handled with dignity — never as a label, never as a marketing angle, always as part of who Stacie is and how the business was built. If a draft fails the brand voice check, this agent returns it with specific notes — not a rewrite, just what's wrong and why. The Draft Writer agents incorporate those notes and resubmit. Use this agent whenever you are unsure if a piece of content sounds right, or whenever you have content that came from outside the system (a guest writer, a brand partner, an AI draft) and needs to be checked.

### 7. Content Calendar Agent

Manages the full posting schedule across Instagram, Facebook, TikTok Business, TikTok Personal, and Stories. It takes approved content, assigns it to a specific date and time, checks for pillar rotation and platform balance, and produces a weekly or monthly calendar view for Stacie's review. It flags conflicts (two booking-push posts in the same week, a gap in TikTok output, a week where The Work pillar hasn't been touched). The calendar is a living document — when events come up, when something needs to be bumped, when Stacie wants to post something spontaneous, the Calendar Agent adjusts and rebalances. Use this agent to plan the week, review the month, or make scheduling decisions.

### 8. Engagement Routing Agent

Monitors and classifies all inbound engagement: comments, DMs, story replies, tags, and mentions. It tiers every inbound contact into a category: Warm Lead (booking potential), Corporate Signal (business or event planner), Partnership/Brand Signal, Organic Follower, or General Community. Each tier gets a different response priority and a different response type. The Engagement Routing Agent does not write replies — it classifies and passes to the Follow-Up Agent. It also flags anything that needs Stacie's personal attention versus something that can be handled with a standard response. Use this agent whenever you need to prioritize your inbox or understand what kind of engagement you're getting.

### 9. Follow-Up Agent

Writes response drafts for comments and DMs, organized by tier. For Warm Leads, it writes a warm but specific reply that moves the conversation toward a booking without pressure. For Corporate Signals, it writes a professional but not stiff reply that opens a conversation and directs them to the right next step. For Partnership/Brand Signals, it writes a reply that is open, curious, and non-committal until more information is gathered. For Organic Followers, it writes a genuine, brief, human response that builds community. All replies go to Stacie for review before sending — the Follow-Up Agent drafts, it does not send. Use this agent to manage your inbox without spending 45 minutes writing replies from scratch.

### 10. Performance Signal Agent

Tracks observable performance signals without requiring formal analytics access. It monitors: which posts get comments with names in them (the animals are becoming characters), which posts generate DMs, which story polls get responses, which weeks produce inbound inquiries, and whether the booking calendar fills faster in months with active campaigns. It is not a metrics dashboard — it is a pattern observer. It produces a weekly signal report (3–5 bullets, no more) that tells Stacie what's working and what to change. Use this agent at the end of each week to check in on what the content is doing.

### 11. Creator Income Agent

Evaluates every piece of content for monetization potential before it posts. It asks: Is this sponsor-friendly? Which brand categories would place well next to this content? Does this work as UGC spec? Is this a pitch asset? It also tracks what content should be produced specifically as creator income inventory — not what to post to the audience, but what to produce for brand pitching. At the end of each month it produces a short brief: what to pitch, to whom, with what assets. Use this agent when building a campaign, reviewing what's been produced, or when you want to know how to turn content into income.

### 12. Corporate Signal Agent

Evaluates content and campaign plans specifically through the lens of corporate buyer credibility. Corporate event planners are not the primary Instagram audience — but they do check Instagram before reaching out. This agent ensures that the website, the Instagram grid, and the Facebook page always have enough professional-grade content that a corporate buyer who lands there via any path has what they need to feel confident. It also monitors for outreach opportunities — companies in the region that might be planning summer events — and drafts initial outreach messages when those opportunities arise. Use this agent when planning a campaign, updating the website, or preparing to pursue corporate business.

### 13. Partnership Signal Agent

Identifies and pursues partnership opportunities with brands, regional organizations, and creators whose audiences overlap with TC's. It evaluates which campaign content is credible enough to support a pitch, drafts initial outreach messages, and tracks what responses come back. It also manages the distinction between a real partnership (mutual value, audience alignment, dignified brand fit) and an opportunity that isn't worth Stacie's time or identity. Not every brand that reaches out is the right brand. This agent helps make that call. Use this agent when a brand reaches out, when you want to pursue a partnership proactively, or when you want to evaluate whether an opportunity is worth pursuing.

## Approval Flow

All content moves through the same stages. Nothing posts without Stacie's eyes on it.

**Stage 1 — Draft**
Content is drafted by the appropriate Draft Writer Agent. At this stage it exists as a text document in `content/drafts/` with a filename by date and platform: `2026-04-08-instagram-meet-duke.md`.

**Stage 2 — Brand Review**
The Brand Voice Agent reviews the draft against TC's non-negotiables. If it passes, it is marked "Brand Approved" in the draft file. If it doesn't pass, specific revision notes are added and it returns to the Draft Writer Agent.

**Stage 3 — Calendar Placement**
The Content Calendar Agent assigns the approved draft a posting date, time, and platform slot. It checks for pillar rotation and platform balance before confirming placement.

**Stage 4 — Stacie Review**
Approved, scheduled drafts are presented to Stacie for final review. She can approve as-is, request edits, bump the date, or pull the post entirely. Nothing moves to posting without explicit approval.

**Stage 5 — Post**
Content is posted on the scheduled date and time. Stories are posted manually; feed posts can be scheduled through platform-native scheduling tools or a third-party scheduler.

**Stage 6 — Follow-Up**
After posting, the Engagement Routing Agent begins monitoring inbound responses. Within 60–90 minutes of posting, Stacie reviews any early comments and sends replies drafted by the Follow-Up Agent. DMs get replied to within 24 hours using the tiered response system.

**Stage 7 — Signal Logging**
The Performance Signal Agent logs the post's observable performance at 48 hours. What engagement came in, what type of comments, any DMs generated, any story responses, any shares. This log feeds the weekly signal report.

## Website Integration

The website is not separate from the content operation — it is the destination that makes the content operation worth running.

Every piece of content either:
1. **Sends people somewhere on the website** (booking page, services page, contact form), or
2. **Sets expectations that the website must then meet** (if the Instagram post describes a specific service, the website page for that service must match what was described), or
3. **Produces content that should also appear on the website** (event recaps, guest testimonials, behind-the-scenes photography that builds trust)

**How website decisions are made in the system:**

- When the Content Strategy Agent plans a campaign, it notes which website pages the campaign touches and whether those pages are ready.
- When the Brand Voice Agent reviews a draft that contains a booking CTA, it checks that the CTA points to a real, working, accurate destination.
- When the Corporate Signal Agent reviews campaign content, it evaluates whether the website has what a corporate buyer would need to take action.
- When the Partnership Signal Agent evaluates a pitch, it confirms that the website reflects a brand worth partnering with.

**Website updates are treated as content tasks.** If a campaign requires a landing page update, that update is assigned, drafted, and approved through the same flow as a post.

## Creator Income Integration

Every piece of content produced in this system is evaluated through two lenses simultaneously: what it does for the audience, and what it does for creator income.

The Creator Income Agent sits alongside the content flow — it does not slow content down or change it for commercial purposes, but it flags opportunities and tracks inventory.

**How it works in practice:**

- A morning routine video gets flagged: "This is sponsor-friendly for western apparel or outdoor lifestyle brands. Produce a clean version with [product category] in frame as a UGC spec asset."
- A herd introduction carousel gets flagged: "This aesthetic matches Ariat and Kimes Ranch visual language. Pull 3 screenshots for a pitch deck."
- A "behind the work" event prep Reel gets flagged: "This demonstrates event credibility for corporate buyers. This is also pitch content for instant photo delivery or event tech sponsors."

At the end of each month, the Creator Income Agent produces a one-page brief: what was produced, what is sponsor-ready, who to pitch, and what to produce next month specifically for income inventory.

**The principle:** Stacie does not produce content for brands at the expense of her audience. She produces content for her audience that happens to be exactly what brands want to be near. The Creator Income Agent's job is to make sure she captures value from that, not just build a sponsor's audience for free.

## Engagement Routing

Every inbound message — comment, DM, story reply, tag, mention — goes through the Engagement Routing Agent before any response is drafted.

**The five tiers:**

**Tier 1 — Warm Lead (Booking Potential)**
Signal: Asks about booking, pricing, dates, availability, what services exist. Asks "do you do X" or "are you available for Y."
Response priority: Same day. Draft a specific, warm reply that moves toward a real conversation without pressure.

**Tier 2 — Corporate Signal**
Signal: Language suggests event planning, company function, team event, client appreciation, or corporate gathering. Email domain of an organization. Mentions "our company" or "for a group."
Response priority: Same day. Professional but not stiff. Open a conversation, direct to the corporate events page, offer a real conversation.

**Tier 3 — Partnership/Brand Signal**
Signal: "I'd love to work with you," "I think there's something here," brand account reaches out, agency inquiry.
Response priority: Within 24 hours. Open but non-committal. Gather information before expressing interest.

**Tier 4 — Organic Follower**
Signal: General positive comment, question about the animals, reaction to a story, shares or saves.
Response priority: Within 48 hours. Brief, genuine, human. Not a marketing response.

**Tier 5 — General Community**
Signal: Local community member, someone who knows Stacie personally, regional neighbor.
Response priority: When time allows. Conversational, personal.

All reply drafts go to Stacie before sending. The system handles classification and drafting — Stacie handles the final voice.

## Weekly Operating Rhythm

The system is designed to require 30 minutes or less per day from Stacie. Here is what that looks like day by day.

**Monday — 20 minutes**
Review the weekly content calendar. Confirm posts are approved for the week. Check the weekend's inbound engagement and approve any reply drafts the Follow-Up Agent has prepared. Flag anything that needs more than a standard reply.

**Tuesday — 10 minutes**
Check in on Monday's post performance. Anything notable? Any DMs from it? Log a quick note. Approve Tuesday's post if it hasn't gone out yet.

**Wednesday — 15 minutes**
Mid-week content check. If something happened — an event, a great horse moment, something the land did — flag it for the Content Strategy Agent and get a draft going. Review and approve anything in the queue.

**Thursday — 10 minutes**
Approve and review Thursday content. Check story engagement from the week so far. If story polls or questions have come in, the Follow-Up Agent has drafts for those responses.

**Friday — 20 minutes**
Review the previous week's content and note what performed. The Performance Signal Agent has a weekly signal report ready. Read it, note anything to change, flag any patterns. Begin approving next week's content queue if it's ready.

**Saturday — 5 minutes**
Check DMs and engagement. Approve any reply drafts. Weekend DMs from families often come in here — warm lead territory.

**Sunday — 10 minutes**
Plan the week. Review what's in the calendar for the coming week. Anything missing? Any events coming up that need content? Flag to the Content Strategy Agent for Monday routing.

**Total: approximately 90 minutes per week across seven days.**

## A Note on How This System Scales

The system as described is designed for where Tribal Cowboy is now: one person, one herd, one operating location, one primary market. As the business grows — more events, more markets, more services — the system scales by adding content volume and adjusting agent outputs, not by rebuilding the architecture.

The Orchestrator stays the same. The approval flow stays the same. The voice rules stay the same. What changes is how much goes through the system, and which opportunities the Creator Income and Partnership Signal agents flag as worth pursuing.

The discipline the system enforces — no corporate language, no clichés, no pressure tactics, Indigenous identity handled with dignity, specificity over generality — those rules do not change with scale. They are what makes the brand worth building.
