## TROUBLESHOOTING

**Q: The buttons don't link to Acuity correctly**  
A: Make sure the services are set up in Acuity first. See ACUITY-SERVICES-SPEC.md for detailed Acuity setup instructions. Once services are live in Acuity, copy the correct booking page URLs and paste them into the button links.

**Q: The page looks different from how I set it up**  
A: Squarespace may have default styling. You may need to:
- Adjust card widths/spacing through the page settings
- Use custom CSS if Squarespace doesn't allow color customization for buttons
- Check that the theme supports the colors you're trying to use

**Q: How do I change the button colors?**  
A: In the Squarespace page editor:
1. Click on a button
2. Look for color/styling options in the right panel
3. Set the color to the hex code provided (e.g., #D4A843)
4. Repeat for each button

**Q: Can I reorder the cards later?**  
A: Yes. The order suggested is:
1. Premium (3 cards) — highest priority
2. Corporate (2 cards) — mid priority
3. Special/Wedding (2 cards) — lower priority
4. Add-ons — optional upgrades

But you can reorder if needed.
