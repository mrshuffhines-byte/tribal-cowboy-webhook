## How to Post

### Instagram Post 1 — Awareness (Week 1)
**Image:** `images/instagram-post-1.png`
**Caption:** Use "Post 1 — Soft Pitch" from campaign draft
**Hashtags:** included in draft
**Best time:** Tuesday or Wednesday, 9–11am or 6–8pm

### Instagram Post 2 — Direct Value (Week 3)
**Image:** `images/instagram-post-1.png` (same image, different caption)
**Caption:** Use "Post 2 — Direct Value" from campaign draft
**Alternate:** Use `images/facebook-header.png` cropped to square for variety

### Instagram Post 3 — Story (Week 4)
**Image:** `images/instagram-post-1.png` or any real horse event photo you have
**Caption:** Use "Post 3 — Storytelling" from campaign draft

### Instagram Story Sequence A (Week 1, after feed post)
**Background slides:** `images/story-background-1.png`
**Mockup slide:** `images/story-photo-mockup.png`
**Copy:** Type each slide's text directly on the image in Instagram Stories editor
- Use white text on the dark background
- Use gold text (#D4A843) for key phrases if your Stories editor supports hex

### Instagram Story Sequence B (Week 3)
**Background:** `images/story-background-1.png` for all slides
**Copy:** Use Sequence B slides from campaign draft

### Facebook Post (Week 2)
**Image:** `images/facebook-header.png`
**Caption:** Use Facebook post from campaign draft
**Post to:** Your business page first, then share to any North Idaho real estate groups you're in (post once, don't spam)

### TikTok (Week 2)
**Cover frame:** `images/tiktok-cover.png` — set as video thumbnail
**Script:** Use TikTok script from campaign draft
**Record:** Talking-to-camera, 30–45 seconds, outdoors near the horses if possible
**Hashtags:** included in draft

### LinkedIn Post (Week 4)
**No image needed** — text-only performs better on LinkedIn
**Post as:** Stacie personally (not the business page)
**Copy:** Use LinkedIn post from campaign draft

## DM Outreach

Send 5–10 DMs per week to real estate agents who post about client events.
Use the DM template from `../2026-04-15-real-estate-social-campaign.md`.

**Do not** follow up more than once if no response.

## Notes

- Lead every conversation with: "Every photo has your brokerage name on it."
- Summer books fast. Use low-pressure language but mention availability honestly.
- If a broker responds to a DM with interest, send `../North40_Proposal.pdf` as a reference point.
- Never post the same pillar two days in a row on Instagram.
